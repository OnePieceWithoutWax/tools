# windows/

PowerShell and batch scripts that aren't Python packages, tracked as plain files.

Each subfolder gets a short README explaining what its scripts do and how to use them.

## Index

- [remote-dev/](remote-dev/README.md) — one-shot setup + health check for SSH/Tailscale/WSL/mosh/tmux remote dev, with guides
- [shell-launchers/](shell-launchers/README.md) — "open shell here" launchers for cmd, Git Bash (plain or with `.venv` auto-activate), and PowerShell
- [jupyter/](jupyter/README.md) — `list-kernel-ipykernel-versions.ps1`, a kernelspec/ipykernel version diagnostic
